# Solar System Demo / 太阳系演示

基于 Three.js 构建的交互式 3D 太阳系可视化模型。

## 打开方式

直接用浏览器打开 `index.html` 即可运行（需要网络加载 Three.js CDN）。

## 操作说明

| 操作 | 效果 |
|------|------|
| 鼠标左键拖拽 | 旋转视角 |
| 滚轮 | 放大 / 缩小 |
| 鼠标右键拖拽 | 平移视角 |
| 键盘 `R` | 重置视角 |

## 模型内容

- **太阳** — 发光球体 + 光晕粒子（Sprite），作为点光源
- **八大行星** — 水星 → 海王星，含公转/自转动画
- **轨道环** — 每个行星对应半透明圆形轨道
- **土星光环** — RingGeometry 环状几何体
- **小行星带** — 火星与木星之间，约 3500 粒子，缓慢公转
- **柯伊伯带** — 海王星之外，约 6000 粒子，极慢公转
- **彗星** — 3 颗椭圆轨道彗星，动态彗尾（靠近太阳时变亮变长）
- **星空背景** — 约 12000 颗随机分布的恒星
- **名称标签** — CSS2DRenderer 实现的文字标签

## 技术栈

- [Three.js](https://threejs.org/) r0.160.0 — 3D 渲染引擎
- OrbitControls — 轨道控制（旋转/缩放/平移）
- CSS2DRenderer — 文字标签渲染
- ES Module + Import Map — 无需构建工具

## 实现要点

- 彗尾使用 `ConeGeometry`，通过四元数实时朝向背离太阳方向
- 彗尾长度和透明度根据彗星与太阳距离动态变化
- 小行星带和柯伊伯带粒子逐帧更新角度实现公转
- 所有粒子使用 `Points` 而非独立 Mesh，保证性能
- 行星轨道使用 `LineLoop`，椭圆轨道使用 `BufferGeometry.setFromPoints`
